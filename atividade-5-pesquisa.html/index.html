Elemento HTML <b>

O elemento <b> serve para deixar o texto em negrito. Ele é usado quando queremos dar destaque visual a uma parte do texto, mas sem dizer que é mais importante.

Exemplo:

<p>Este é um texto com uma palavra em <b>negrito</b>.</p>

Este é um texto com uma palavra em negrito.
